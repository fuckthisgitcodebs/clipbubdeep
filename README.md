
ClipBub Deep – Next‑Gen Text Command Overlay

A sleek, keyboard‑themed accessibility service that gives you instant access to text actions and clipboard history.

Features

· Radial menu appears when you select text – fully customizable actions.
· Swipe gestures on main button: up/down for extra functions.
· Clipboard history – stores up to 50MB of text entries locally, searchable and filterable.
· Keyboard‑inspired UI with SwiftKey Neon theme (dark background, cyan accents).
· Tasker integration via broadcast intents.

How to Use

See in‑app help (tap the Help button) for detailed instructions.

Building

· Use Android Studio or run ./gradlew assembleRelease.
· GitHub Actions automatically builds on push.
